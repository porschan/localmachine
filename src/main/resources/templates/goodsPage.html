API食用方法：

获取全部商品

URL:http://localhost:8080/localmachine/rest/allgoods

method:GET

type:rest

return type:json style




获取单个商品

URL:http://localhost:8080/localmachine/rest/goods/1

method:GET

type:rest

return type:json style




更新商品

URL:http://localhost:8080/localmachine/rest/goods?id=1&goodsName=iphone xr

method:PUT

type:rest

return type:json style




保存商品

URL:http://localhost:8080/localmachine/rest/goods?goodsName=iphone xs

method:POST

type:rest

return type:json style




删除用户

URL:http://localhost:8080/localmachine/rest/goods/6

method:DELETE

type:rest

return type:json style
