API食用方法：

获取全部用户

URL:http://localhost:8080/localmachine/rest/users

method:GET

type:rest

return type:json style




获取单个用户

URL:http://localhost:8080/localmachine/rest/user/1001

method:GET

type:rest

return type:json style




更新用户

URL:http://localhost:8080/localmachine/rest/user?id=1001&lastName=porschan&email=710437653@qq.com&gender=1&department.id=1

method:PUT

type:rest

return type:json style




保存用户

URL:http://localhost:8080/localmachine/rest/user?lastName=john&email=710437653@qq.com&department.id=104

method:POST

type:rest

return type:json style




删除用户

URL:http://localhost:8080/localmachine/rest/user/1001

method:DELETE

type:rest

return type:json style
